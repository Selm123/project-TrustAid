# TrustAid Frontend

Vite + React + TypeScript frontend for **TrustAid**.

## Quick Start
```bash
# 1) (optional) set API base
cp .env.example .env   # edit VITE_API_BASE if needed

# 2) run (mac/linux)
./run.sh

# or on Windows:
run.bat
```
The app defaults to `http://localhost:8000`. You can toggle **Demo Mode** in the left sidebar to run without backend.
